# Описание класса Service
Класс для работы с услугами тату-салона.

## Атрибуты

* Name:String
* ID:Int
* Time:Integer
* Price:Double

## Описание атрибутов

* Name:String - название услуги.
* ID:Int - идентификатор в БД.
* Time:Integer - количество времени, требующегося для услуги.
* Price:Double - цена.
